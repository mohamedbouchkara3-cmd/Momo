## Project Overview
This project is designed to showcase the various juices available at Tarik Juice Bar, providing an easy-to-navigate website for customers.

## Features
- Interactive menu displaying all juice options.
- User-friendly design and responsive layout.
- Contact form for customer inquiries.

## Installation
To install the project locally, please follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/mohamedbouchkara3-cmd/Momo.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Momo
   ```
3. Open the `index.html` file in your browser.

## Usage
- Simply access the website via your browser to view the menu and contact options.
